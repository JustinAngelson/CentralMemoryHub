API Documentation
Authentication

All API endpoints require the X-API-KEY header with your API key.

Structured Memory
  • POST /memory/structured - Add a new structured memory entry
  • GET /memory/structured/{id} - Retrieve a structured memory entry by ID
Unstructured Memory
  • POST /memory/unstructured - Add a new unstructured memory entry
  • GET /memory/unstructured/{id} - Retrieve an unstructured memory entry by ID
Search
  • POST /search - Search for similar unstructured data
Shared Contexts
  • POST /context - Add a new shared context
  • GET /context - Retrieve all shared contexts
